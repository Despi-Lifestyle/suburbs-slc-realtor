Data Reference

What Gets Captured Per Lead

Understanding the difference between form data and analytics data.

Formspree / Lofty CRM

Form Data

Captured when someone fills out the lead form on the Relocation Guide page:

  • firstName — First name (required)
  • email — Email address (required)
  • phone — Phone number (required)
  • _subject — "New Lead: Relocation Guide Download"

The form captures contact info only. It does NOT capture UTM parameters.

Google Analytics

Analytics Data

Tracked automatically when someone clicks a UTM link:

  • utm_source — Where they came from
  • utm_medium — Type of link placement
  • utm_campaign — Specific campaign
  • Landing page URL, time on site, pages viewed

View in GA4: Acquisition → Traffic Acquisition report.

Standards

UTM Naming Conventions

Follow these rules so your analytics data stays clean and consistent.

Rules

  • Always use lowercase — UTM params are case-sensitive
  • Use dashes (-) instead of spaces or underscores
  • No special characters — no &, ?, =, #, or %
  • Keep it short but descriptive — you'll be reading these in GA reports
  • Be consistent — always use the same value for the same source/medium

Standard Values

ParameterPurposeStandard Values
utm_sourceWhere traffic comes fromyoutube, instagram, facebook, tiktok, email, google, direct
utm_mediumType of link placementvideo-description, channel-bio, pinned-comment, story, post, email-signature, newsletter, qr-code
utm_campaignSpecific campaign or contentFree-form. Examples: draper-suburb-tour, market-update-q1-2026

Analytics Setup

Google Analytics Configuration

One-time setup to start tracking UTM data across the site.

Step 1: Get Your GA4 Measurement ID

  1. Go to Google Analytics
  2. Admin → Data Streams → Web
  3. Copy your Measurement ID (format: G-XXXXXXXXXX)

Step 2: Add the Tracking Snippet

Add this code to every HTML page, inside <head>, before the closing </head> tag:

<!-- Google Analytics --> <script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){dataLayer.push(arguments);} gtag('js', new Date()); gtag('config', 'GA_MEASUREMENT_ID'); </script>

Replace GA_MEASUREMENT_ID with your actual ID on every page.

Step 3: Verify It Works

  1. Open your site in a browser
  2. Open GA4 → Reports → Realtime
  3. You should see yourself as an active user
  4. Click a UTM link — check that source/medium/campaign appear in Realtime

Step 4: View UTM Data

In GA4: Acquisition → Traffic Acquisition

Look at the Session source/medium and Session campaign columns.
